# 👉[가자서울 바로가기](http://가자서울.shop)  


## 💻 프로젝트 소개

 go-seoul은 서울 여행지 공유 사이트 입니다.       
 서울 여행의 숨은 명소들의 사진을 올리며 서로 공유할 수 있습니다.

***
## ⚛️ 기능  
* 구글 로그인 & 네이버 로그인 기능 구현      
* 파일첨부 게시판 및 썸네일 저장 기능       
* Ajax를 이용한 댓글 처리 및 Starrr 라이브러리를 이용한 별점 기능         
* 지역별 정렬 및 페이징 구현            
* AWS를 이용하여 서버 배포
